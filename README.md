# Tokyo Revengers Character & Gang Explorer

A responsive web application built with **Flask**, **MySQL**, **HTML**, and **CSS**, designed to showcase the world of **Tokyo Revengers**.  
The application allows users to:

- Browse characters and their associated gangs
- Filter and search characters dynamically
- Explore episode information (title, description, arc, rating, art)
- Submit their own details to join a selected gang
- Enjoy a visually rich UI with anime-styled effects and glassmorphism displays

---

## Features

|      Feature       |                           Description                                  |
|--------------------|------------------------------------------------------------------------|
| Character Database | View character cards with images, descriptions, and gang affiliations. |
| Search & Filtering | Filter characters by gang or search by name.                           |
| Episode Archive    | Browse detailed episode summaries with thumbnails.                     |
| Join a Gang Form   | Users submit their name, email, favorite character, and selected gang. |
| Modern UI          | Dark theme, animated elements, and polished visual style.              |

---
